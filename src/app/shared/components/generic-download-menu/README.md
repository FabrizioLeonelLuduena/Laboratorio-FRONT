# GenericDownloadMenuComponent

## Descripción
El **GenericDownloadMenuComponent** es un componente genérico reutilizable que muestra un **menú desplegable de descarga**, ofreciendo opciones para exportar información en **Excel o PDF**.  
Está diseñado para integrarse fácilmente en toolbars, tablas u otras vistas donde se requiera la exportación de datos.
